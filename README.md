# tutoriel version captures d'écrans

Dans ce tutoriel, il y a plusieurs points importants à retenir:

⚠️ tout les fichiers qui sont mis à disposition sur ce Github sont 100% SAFE et LEDGIT, ne vous ammusez pas à installer d'autres fichiers ! ⚠️

______________________

Liste des logiciels à installer :
- REBOOT launcher (version 5.2)
- Une/des saisons de fortnite
- node.js
- lawinserver (v1 ou v3)
- free downloads manager

______________________

liste des DLL à mettre dans votre lanceur reboot

launcher reboot -> ⚙️Settings

- Reboot DLL : Reboot_Ultimate.dll
- Console DLL : Fortnite_Console_v2.1.dll
- Base DLL : craniumv2.dll
